1. Download the 'turialspoint.js' file.
2. Make sure you have 'Node' installed on your machine to run the javascript file.(if not, install from https://nodejs.org/)
3. Open command prompt or terminal
4. Move to the directiry in which you downloaded the 'turialspoint.js'.(use 'cd' command to navigate in command prompt/terminal).
5. Type the following command to download the tutorial :
	node turialspoint --url="URL_OF_NOTES_HOME_PAGE_ON_TUTORIALSPOINT"

	URL_OF_NOTES_HOME_PAGE_ON_TUTORIALSPOINT should be replaced with the url of notes you want to download.
